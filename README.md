# 🚀 CheckMate CV/ML

Проект для обработки документов с использованием методов Computer Vision и Machine Learning.
Выполнен в рамках обучения в компании Singularis
---

## 📌 Описание

Данный проект выполняет обработку и анализ документов с применением ML-моделей и OCR методов.
Поддерживается запуск как локально, так и через Docker.
Настройка параметров и типов обработки происходит в env файле.

---

## ⚙️ Возможности

* 📄 Обработка документов
* 🤖 Использование ML-моделей
* 🐳 Поддержка Docker
* 🔧 Гибкая настройка через конфигурации

---

## 📦 Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/Zerrant2/CheckMate-CV_ML.git
```

---

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

---

## 🧠 Модель

Модель не включена в репозиторий из-за ограничения GitHub на размер файлов.

📥 Скачать можно здесь:
👉 **[Google Drive ссылка на модель](https://drive.google.com/file/d/1s36cDQfDKOz26NHiEY99tT7MxYhe1lJ9/view?usp=sharing)**

После скачивания помести файл в:

```bash
app/DocRes/checkpoints/
```

---

## 🐳 Запуск через Docker

### Сборка образа

```bash
docker build -t checkmate-cv .
```

### Запуск контейнера

```bash
docker run -p 8000:8000 checkmate-cv
```

---

## ▶️ Запуск проекта осуществляется через Docker и RabbitMQ

## ⚠️ Важно

* Файлы моделей (*.pkl) не хранятся в репозитории
* Используй `.gitignore`, чтобы избежать загрузки больших файлов
